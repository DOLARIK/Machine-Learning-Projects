***How to Run the Code***
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
-Open MATLAB
-Open my project folder in the ‘Current Folder Tab’
-Open the Script MNISTPredict.m and run it.
-----The program will ask a couple of questions. (like Dataset Location and Hyperparameter Values)
-----Kindly put all the dataset in one folder.
-----Then based upon the answers provided by the user, It will start the Training. 
-----After Training ends it will feed the learned parameters to the Test Model.
-----After the Model has generated the predictions, the program will present a slideshow of 100 random test images and their predictions. 
-To see the Slideshow again separately the user can run the script funrunMNIST.m (This script will also present 100 random Test Images and their Predictions)
